# MioriticMinds frontend

## Getting Started

### Prerequisites

* [Node.js](https://nodejs.org/en/)

### Installing

1. `git clone git@github.com:edwardinio18/proiect-colectiv-frontend.git`
2. `npm install`
3. `npm run dev`
4. Open `http://localhost:5173` in your browser
5. Enjoy!
